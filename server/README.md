# Linkodkod Server

## API Documentation

### Post Endpoints

| Method | Endpoint          | Description                   |
| ------ | ----------------- | ----------------------------- |
| POST   | `/posts/`         | Create a new post             |
| GET    | `/posts/`         | Get all posts                 |
| GET    | `/posts/:id`      | Get post by id                |
| PUT    | `/posts/:id`      | Update an existing post by id |
| DELETE | `/posts/:id`      | Delete post by id             |
